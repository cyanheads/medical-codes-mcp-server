# 0.1.5 — 2026-06-20

## Changed

- **`@cyanheads/mcp-ts-core`** `^0.10.6` → `^0.10.9`. No server code changes — adopts three upstream patch releases (framework changelogs [0.10.7](https://github.com/cyanheads/mcp-ts-core/blob/main/changelog/0.10.x/0.10.7.md), [0.10.8](https://github.com/cyanheads/mcp-ts-core/blob/main/changelog/0.10.x/0.10.8.md), [0.10.9](https://github.com/cyanheads/mcp-ts-core/blob/main/changelog/0.10.x/0.10.9.md)). Transitive bumps pulled in via the lock: `@hono/node-server` `2.0.4` → `2.0.5`, `hono` `4.12.25` → `4.12.26`.
- **`scripts/check-dependency-specifiers.ts`** (new) — synced from framework 0.10.9; registered in `devcheck` as the `Dependency Specifiers` step, hard-failing on floating dependency specifiers (`latest`, `*`, pre-release dist-tags) in `package.json` and `bun.lock`'s workspace map. ([cyanheads/mcp-ts-core#246](https://github.com/cyanheads/mcp-ts-core/issues/246))
- **`devcheck.config.json`** — added `packaging.pluginManifests: true`, enabling the new plugin-marketplace manifest checks in `lint:packaging` (validates `.claude-plugin/plugin.json` and `.codex-plugin/` descriptions, display fields, and install args). ([cyanheads/mcp-ts-core#240](https://github.com/cyanheads/mcp-ts-core/issues/240))
- **`scripts/devcheck.ts`, `scripts/build-changelog.ts`, `scripts/check-framework-antipatterns.ts`, `scripts/check-skill-versions.ts`, `scripts/lint-packaging.ts`** — re-synced from the framework: fresh-scaffold git-repo guards on git-dependent checks ([cyanheads/mcp-ts-core#242](https://github.com/cyanheads/mcp-ts-core/issues/242), [#243](https://github.com/cyanheads/mcp-ts-core/issues/243)), worktree-deletion guard in `check-skill-versions` ([cyanheads/mcp-ts-core#237](https://github.com/cyanheads/mcp-ts-core/issues/237)), and the plugin-manifest checks in `lint-packaging` ([cyanheads/mcp-ts-core#240](https://github.com/cyanheads/mcp-ts-core/issues/240)).
- **Framework skills re-synced** — `api-context` documents the `ctx.content` media collector ([cyanheads/mcp-ts-core#239](https://github.com/cyanheads/mcp-ts-core/issues/239)); `git-wrapup` v1.4, `orchestrations` v1.4, and `polish-docs-meta` updated; `api-auth`, `api-canvas`, `api-config`, `api-errors`, `api-services`, `api-telemetry`, `field-test`, `report-issue-local`, `tool-defs-analysis`, and `add-tool` carry upstream body/version drift.
